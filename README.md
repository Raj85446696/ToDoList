# 📝 To-Do List App

A simple To-Do List application built using Node.js, Express.js, and EJS templating engine.

---

## 🚀 Features

- Add new tasks
- View list of tasks
- Delete tasks (optional enhancement)
- Styled with basic CSS

---

- Node.js
